# Tech Stack

- Frontend: Next.js (App Router) with React 19 and TypeScript
- Backend: Fastify 5.x
- State: Redux Toolkit
- Styling: Tailwind CSS + Radix UI primitives
- Forms/Validation: React Hook Form + Zod
- Database: SQLite
- Testing: React Testing Library
- Tooling: pnpm, ESLint, TypeScript (strict)
